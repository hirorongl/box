カテゴリ列挙のテーマ変数
　カテゴリ名称を表示　　categories,categorieslink,categorieslinkcode,categorieslinkid
　　　　　　　　group_id, group_code, grouplink_id, grouplink_code
　　　　　　　　区切りはLANG_USERBOX['category_separater']で指定しています。
　カテゴリコードを表示　group_id_code, group_code_code, grouplink_id_code, grouplink_code_code
　　　　　　　　区切りはLANG_USERBOX['category_separater_code']で指定しています。
featuredimage 最初に表示される画像（ファイル保存）タイプのアトリビュート（追加項目）
id 管理画面で表示されているid 1,2,3,4,5,...(システムで指定された数字)
code 管理画面で指定したcodeの値
