概要

User 用の項目を自由に追加するプラグインです。

機能

  • データ毎にfeaturedimageの登録が可能：追加項目で、最初に表示される画像（ファイル保存）タイプの項目
  • 画像(featuredimage)付カテゴリ別フィード発行
  • ogpメタタグ編集（個別データ表示、カテゴリ別件数一覧、カテゴリ別一覧）
  • カスタム設定：各サイトで独自の言語ファイル、テーマ設定が可能
  • カテゴリ別にテーマ設定が可能

関連プラグイン

  • jQuery 1.3以上のバージョン：jQeryプラグインが有効な場合にのみdate pickerが動作します。
    jQueryのロード順はUserBoxより前にしてください。

制限事項

  1. 言語ファイルは日本語のみ同梱しています。（既定値は日本語になっています）
  2. 文字コードはutfのみ対応しています。
  3. mySQLのみ対応しています。
  4. 主にXrea仕様 および SaaSes サーバで動作作検証しています。

注意

同一サイトにDataBoxがある場合、バージョン（幹番）は同じである必要があります。
