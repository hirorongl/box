DataBox プラグイン

自動タグ(2012/02/24 update)

書式 説明
[databox:count カテゴリidまたはカテゴリコード] データ件数
例１：[databox:count]
例２：[databox:count category_id:1]
例３：[databox:count category_code:AAA]
例４：[databox:count category_code:AAA|BBB] カテゴリコードがAAA またはBBB
例５：[databox:count category_code:AAA|BBB|CCC] カテゴリコードがAAA または BBB または CCC
例６：[databox:count category_code:AAA|BBB mode:and] カテゴリコードがAAA かつ BBB
[databox:newlist　RSSファイルほか例２参照] 新着データ
例１：[databox:newlist]
例２：[databox:newlist rss_file:databox.rss title_trim_length:50 intervalday:70 limitcnt:40 newmarkday:30 templatedir:XXX category_id:99 category_code:XXX]
[databox:data id] データ
例１：[databox:data]
例２：[databox:data id:1 templatedir:XXX nohitmsg:yes]
[databox:category　カテゴリidまたはカテゴリコード　ほか] カテゴリ別一覧
例１：[databox:category]
例２：[databox:category category_id:1 templatedir:XXX nohitmsg:yes perpage:10 page:1 order:id]
例３：[databox:category category_code:AAA ]
例４：[databox:category category_code:AAA|BBB] カテゴリコードがAAA または BBB
例５：[databox:category category_code:AAA|BBB|CCC] カテゴリコードがAAA または BBB または CCC
例５：[databox:category category_code:AAA|BBB mode:and] カテゴリコードがAAA かつ BBB
order に指定できるのは、 random date orderno code title description id と　表示制限されていない追加項目のテーマ変数　です
[databox:additioncount　テンプレートフォルダ　追加項目値] 追加項目別件数一覧
例１：[databox:additioncount]
例２：[databox:additioncount templatesetvar:XXX value:XXX]
[databox:lastmodified　テンプレートフォルダ　追加項目値]
例１：[databox:lastmodified]
例２：[databox:lastmodified category_id:1]
例３：[databox:lastmodified category_code:XXX]
[databox:lastmodified　テンプレートフォルダ　追加項目値]
例１：[databox:lastcreated]
例２：[databox:lastcreated category_id:1]
例３：[databox:lastcreated category_code:XXX]
[databox:field　テンプレートフォルダ　追加項目値　ほか例２参照]
例１：[databox:field field_id:99 value:xx ]
例２：[databox:field field_id:99 value:xx templatedir:x nohitmsg:yes perpage:99 page:99 order:id]
例３：[databox:field field_code:xxx value:xx templatedir:x nohitmsg:yes perpage:99 page:99 order:id]
